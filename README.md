# Sora OS

A minimal 32-bit x86 operating system kernel written in C and Assembly, built from scratch as a university project.

---

## Overview

Sora OS boots via GRUB Multiboot and implements core OS concepts entirely without external libraries — memory management, interrupt handling, preemptive scheduling, multithreading, a PS/2 keyboard driver, and an interactive kernel-space shell.

---

## Features

| Component | Description |
|---|---|
| **Boot Entry** | GRUB Multiboot compliant — no custom MBR needed |
| **GDT** | Global Descriptor Table with kernel code/data segments |
| **IDT** | 256-entry Interrupt Descriptor Table (exceptions + IRQs) |
| **PIC** | 8259A remapping so hardware IRQs don't clash with CPU exceptions |
| **PIT** | 100 Hz timer driving preemptive scheduling |
| **VGA Console** | 80×25 text mode, hardware cursor, scrolling, status bar |
| **PMM** | Manual bitmap allocator — one bit per 4 KB page frame |
| **Scheduler** | Preemptive round-robin with Process Control Blocks |
| **Threads** | Kernel threads sharing parent address space + mutex support |
| **Keyboard** | PS/2 IRQ1 driver with full scancode-to-ASCII table |
| **Serial** | COM1 UART debug output at 9600 baud |
| **Shell** | Interactive kernel-space command interpreter |
| **Panic** | Kernel panic handler — white on red, halts CPU |

## Project Structure

```
SoraOS/
├── src/                    # All source files
│   ├── boot.asm            # GRUB Multiboot entry point (kernel_start)
│   ├── kernel_main.c       # Kernel entry, boot screen, init sequence
│   ├── gdt.c               # Global Descriptor Table setup
│   ├── gdt_flush.asm       # lgdt + segment register reload
│   ├── idt.c               # IDT setup and IRQ dispatch
│   ├── isr.asm             # ISR stubs for vectors 0–47
│   ├── pic.c               # 8259A PIC driver
│   ├── pit.c               # PIT timer driver (100 Hz)
│   ├── vga.c               # VGA text-mode console driver
│   ├── pmm.c               # Physical memory manager (bitmap)
│   ├── scheduler.c         # Preemptive round-robin scheduler
│   ├── context_switch.asm  # ESP save/restore for context switch
│   ├── thread.c            # Kernel thread management
│   ├── mutex.c             # Mutex with x86 test-and-set
│   ├── keyboard.c          # PS/2 keyboard driver
│   ├── serial.c            # COM1 serial driver
│   ├── shell.c             # Interactive kernel shell
│   └── kernel.c            # Kernel panic handler
├── include/                # Header files for all modules
├── build/                  # Object files — auto-generated by make
├── iso/                    # GRUB ISO structure (for make iso)
├── linker.ld               # Linker script — kernel at 0x100000
├── Makefile                # Build system
└── kernel.bin              # Final kernel ELF binary (after make)
```

## Requirements

Run on **Windows 11 with WSL** (Debian/Ubuntu):

```bash
sudo apt update
sudo apt install -y gcc-i686-linux-gnu binutils-i686-linux-gnu \
                   nasm qemu-system-x86 grub-pc-bin xorriso
```

---

## Build and Run

```bash
# Build the kernel
make

# Run in QEMU (serial debug output appears in this terminal)
make run

# Debug — QEMU paused, GDB stub on port 1234
make debug

# Build a bootable ISO image
make iso

# Remove all build artifacts
make clean
```

**GDB debug workflow:**

```bash
# Terminal 1
make debug

# Terminal 2
i686-linux-gnu-gdb kernel.bin
(gdb) target remote localhost:1234
(gdb) break kernel_main
(gdb) continue
```

## Shell Commands

| Command | Description |
|---|---|
| `help` | List all available commands |
| `clear` | Clear the screen |
| `echo <text>` | Print text to the screen |
| `time` | Show system uptime (hours, minutes, seconds) |
| `ps` | List running processes with state and priority |
| `threads` | List kernel threads and their parent process |
| `meminfo` | Show total, used, and free memory |
| `reboot` | Reboot via keyboard controller reset |
| `halt` | Disable interrupts and halt the CPU |
| `about` | About Sora OS |
| `banner` | Display the Sora OS ASCII banner |

## Memory Layout

```
0x00000000 – 0x000FFFFF   Low memory (BIOS, VGA buffer at 0xB8000)
0x00100000                Kernel load address (1 MB mark)
  [.multiboot]            Multiboot header + kernel_start entry code
  [.text]                 Kernel code
  [.rodata]               Read-only data (scancode tables, exception names)
  [.data]                 Initialized data
  [.bss]                  Zero-initialized (PMM bitmap, stack, PCB pool)
[kernel_end]              Start of free physical pages managed by PMM
```

## Platform

- **Host** — Windows 11, WSL (Debian)
- **Target** — x86 (i686) 32-bit protected mode
- **Bootloader** — GRUB2 + Multiboot specification
- **Emulator** — QEMU `qemu-system-i386`
- **Compiler** — `i686-linux-gnu-gcc` (freestanding, no stdlib)
- **Assembler** — NASM (ELF32 output)
